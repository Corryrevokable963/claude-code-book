## 🧭 What This Is

`claude-code-book` is a Chinese book about Claude Code and agent systems. It breaks down how an AI agent works from the inside. It focuses on the harness layer, the message loop, tool use, and the parts that help an agent think and act.

This book is built for readers who want a clear path through a hard topic. It covers the shape of an Agent Harness, how Claude Code fits into it, and how you can use the same ideas to build your own agent.
